# TP React - Gestion d'état et structuration des composants

## Objectif
Ce TP permet de découvrir :
- La création de formulaires simples (contrôlés et non-contrôlés)
- Le partage de données entre composants
- L'utilisation du contexte React pour gérer des informations globales

##  Fonctionnalités
-  Formulaire contrôlé avec useState
-  Formulaire non-contrôlé avec useRef
-  Convertisseur de température bidirectionnel
-  Système de connexion/déconnexion avec Context API
-  Design moderne et responsive

##  Technologies utilisées
- React 18
- CSS3 avec animations
- Hooks React (useState, useContext, useRef)
- Context API

 Ouvrir [http://localhost:3000](http://localhost:3000) pour voir l'application.

##  Structure du projet

```
src/
├── components/
│   ├── FormulaireControle.js      # Formulaire contrôlé
│   ├── FormulaireNonControle.js   # Formulaire non-contrôlé
│   ├── TemperatureInput.js        # Champ de saisie de température
│   ├── TemperatureConvertor.js    # Convertisseur de température
│   ├── Profil.js                  # Composant profil utilisateur
│   └── UtilisateurContext.js      # Contexte utilisateur
├── styles/
│   └── App.css                    # Styles CSS
└── App.js                         # Composant principal
```
